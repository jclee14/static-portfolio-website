SBIDU Game

           “SBIDU” or “Sorry, but I don’t understand” was an interactive game which my friends and I made with Code::Blocks software and C++ language, when we were studying computer graphic programming class at our college. The game concept was to challenge the player with 4 missions (hangman, number guessing, secret name, and Rock Paper Scissors). On each stage, playing time and second-chances were limited, hence the player must focus on every single second. Moreover, the game system could record a player’s name by inputting at the welcome page. Players could select 2 difficult-level, and the last but not least, there was score summary at the end of this game.
