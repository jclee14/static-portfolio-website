Doggy Poopy Game

           “Woof woof! My beloved dog likes to poop. However, he is a clean creature, hence let find the toilet for him!”
           “Doggy Poopy” was a Flash game that I created in my computer programming class by using ActionScript 3.0. The player had to control a dog on a flying rocket and order him to poop into the floating toilet accurately. If it was hit, these toilets would disappear. The goal was to clear all flying object in the scene. Moreover, his shit was limit, hence the player should pay attention to it.
